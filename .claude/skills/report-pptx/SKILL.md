# report-pptx

종목 리서치 Markdown을 **투자증권 리서치센터 톤**의 PPTX로 내보내는 스킬입니다.

## 실행

```bash
python3 .claude/skills/report-pptx/build_pptx.py {종목명}
# 또는 경로/날짜 지정
python3 .claude/skills/report-pptx/build_pptx.py reports/삼성전자.md --date 2026-06-16
```

- 입력: `reports/{종목명}.md`
- 출력: `reports/pptx/{종목명}.pptx` (`pptx/` 폴더 없으면 자동 생성)
- 의존: `python-pptx` (필요 시 `python3 -m pip install python-pptx`)

## 슬라이드 순서 (항상 고정)

표지(종목명·작성일) → 종목 개요 → 재무 요약(표+최근 3개년) → 가격/추세(차트) → 뉴스·심리 → 리스크 → 한 줄 종합

`.md`의 섹션 순서와 무관하게 위 순서로 배치됩니다. 섹션은 `##` 제목의 **키워드로 매칭**합니다(예: "재무"·"실적" → 재무 요약, "가격"·"추세"·"차트" → 가격/추세). 매칭되는 섹션이 없으면 "원본 내용 없음"으로 표시됩니다.

## 입력 .md 작성 규칙

- 맨 위 `# 종목명` (H1) 1줄 — 표지 제목이 됩니다.
- `작성일: YYYY-MM-DD` 1줄(선택) — 없으면 `--date` 또는 오늘 날짜.
- 각 슬라이드는 `## 섹션명` 으로 시작. 본문은 문단/불릿(`- `)/표/차트로 구성.
- **표**: 표준 Markdown 표(`| ... |` + 구분행). 첫 열은 라벨(좌측정렬), 나머지는 수치(우측정렬).
- **차트**: `![chart](assets/charts/005930.png)` 또는 `chart: assets/charts/005930.png`. 이미지가 없으면 "차트 이미지 없음" 안내 박스가 들어갑니다(수치 날조 금지).
- **출처/기준일**: 수치가 있는 문장·표 아래에 `(출처: DART, 2025/연간)` 처럼 표기. `(출처...)` 로 시작하는 줄은 작은 회색 각주로 렌더됩니다.
- **키 수치 강조**: 강조할 값을 `**...**` 로 감싸면 한화 오렌지 볼드로 렌더됩니다(문단·불릿·표 셀 모두). 과용하지 말 것.

입력 양식 예시는 `templates/리포트_양식.md` 참고.

## 디자인 (코드에 고정 — 수정 시 build_pptx.py 상단 토큰만)

- 배경 화이트 / 제목 딥네이비 `#1F2A44` / 본문 그레이
- 한화 오렌지 `#F37321` 는 **표지 상단 바·구분선·키 수치 강조**에만 절제 사용
- 표: 헤더행만 딥네이비 배경+흰 글자, 본문행은 얇은 그레이 괘선
- 한글 폰트 **맑은 고딕** 단일 고정(라틴/한글 동일 지정 → 글자 깨짐 방지)
- 표가 슬라이드 밖으로 넘치면 **행을 잘라** "…외 N개 행 생략" 으로 표기

## 가드레일 (코드가 강제)

- **매수/매도·목표가 단정 표현 금지** — `목표주가`, `매수 추천` 등 금지어가 있으면 **생성을 중단**하고 위반 항목을 출력합니다. 판단 근거까지만 남기세요.
- **출처 없는 수치 경고** — 수치가 있는데 `(출처)` 표기가 없는 섹션은 경고를 출력합니다(생성은 진행).
- 표지·종합 슬라이드와 모든 페이지 푸터에 **학습용·투자권유 아님** 문구가 자동으로 들어갑니다.
